# Termax — SSH Terminal 开发计划

> 更新日期: 2026-05-22
> 技术栈: Tauri v2 + Rust + React 19 + xterm.js

---

## 版本路线

- **M0** — 环境搭建 & 脚手架 ✅
- **M1** — MVP: 单个 SSH 连接能跑通 ✅
- **M2** — 多 Tab + 配置管理 ✅
- **M3** — SFTP 文件浏览器 ✅
- **M4** — 主题 & 体验打磨 ✅
- **M5** — 高级功能（端口转发/堡垒机） ⏳ 进行中
- **M6** — 移动端适配 (Android 优先) ⏳ 待开始

---

## 技术架构

```
React 19 + TypeScript ─── Tauri IPC/Event ─── Rust (russh + portable-pty)
  xterm.js (WebGL)                                 ssh/ session/ sftp/
  Zustand (state)                                  storage/ local/
  Tailwind v4
```
